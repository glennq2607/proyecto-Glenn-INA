EJEMPLOS DE ATRIBUTOS GLOBALES

Los atributos globales sonatrbutos comunes a todos los elementos HTML; pueden usarse en todos los elementos, aunque pueden no tener efecto en algunos de ellos.

Los atributos globales pueden especificarse en todos los elementos HTML,e incluso en aquellos no especificados en el stándar. Esto significa que cualquier elemento no estandar también debe permitir estos atributos, aún cuando el uso de tales elementos significa que el documento ya no cumple con HTML5.

accesskey

Proporcionan y genera un acceso de teclado para el elemento actual.

Ir a ejemplos

Class

Especifica uno o más nombre de clases de un elemento(ejemplos los subtítulos color red y font-size 15px)

dir

Especifica la dirección del texto para el contenido de un elemento, observe que el subtítulo dir esta a la derecha.

hidden

Especifica que un elemento no es todavía, o ya no es,relevante,es decir, lo oculta

id

Especifica un identificador único para un elemento ejemplo ver título con color azúl y font-size 20px

lang

Especifica el lenguaje del contenido del elemento ejemplo este párrafo en español

Stile

Especifica una línea de estilo CSS de un elemento ejemplo este párrafo con style color verde

tabindex

Especifica el orden de tabulación de un elemento. El orden comienza en uno.

tabindex

Especifica el orden de tabulación de un elemento. Este tiene dos.

title

Especifica información adicional acerca de un elemento. La información generalmente será mostrada por un globo al poner el puntero encima, por ejemplo al ubicarse sobre este subtitulo aparecerá mensaje "Ejemplo de título"

contenditable

Especifica si el contenido de un elemento es editable o no, por ejemplo al dar clip sobre este texto se podrá editar.

draggable

Especifica si un elemento se puede arrastrar o no, este párrafo se puede arrastrar

spellcheck

Especifica si el elemento ha de tener su ortografía y gramática marcada o no. Los elementos que pueden tener revisión ortográfica son: inputs de tipo texto,textareas y elementos en modo editable, por ejemplo al dar doble clic sobre este párrafo se pueden ver marcas de ortografía.
